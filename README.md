**Instructions**

Part 1: Clone the Repository

	Click the green "Code" button on this repository
	Copy the HTTPS URL
	Open your terminal/command prompt
	Navigate to where you want to store this project
	Run: git clone [paste-your-url-here]
	Navigate into the folder: cd [repository-name]

**Part 2: Complete the Profile**

	Open the file student_profile.md in your favorite text editor
	Fill in your information where indicated
	Save the file

**Part 3: Write Your First Program**

	Open the file hello.py (or create it if it doesn't exist)
	Write a simple program that prints "Hello, GitHub!" and your name
	Test your program by running: python hello.py

**Part 4: Commit Your Changes**

	Check what files you've modified: git status
	Stage your changes: git add student_profile.md hello.py
	Commit with a message: git commit -m "Complete student profile and hello program"

**Part 5: Push to GitHub**

	Push your changes: git push origin main
	Note: Your branch might be called master instead of main
	Go to your repository on GitHub and verify your changes appear


**Common Issues and Solutions**

"Permission denied" error when pushing:

	Make sure you've accepted the assignment properly
	Check that you're pushing to your own repository, not the template

"Remote origin already exists":

	This is normal, just proceed with your work

Changes not showing on GitHub:

	Make sure you committed your changes: git commit
	Make sure you pushed them: git push

**Submission**

	Your work is automatically submitted when you push to GitHub. I can see all your commits and files.
